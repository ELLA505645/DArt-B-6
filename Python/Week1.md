# 1️⃣ 학습 내용 정리

## 1. 자료구조와 순차 자료형

### 개념정리

**튜플**
- 한 번 할당되면 변경될 수 없는, 고정 길이를 갖는 파이썬의 순차 자료형
- 튜플을 생성하는 방법: 쉼표로 구분되는 일련의 값을 괄호로 감싼다
- 튜플에서 값 분리하기: 등호(=) 오른쪽에 있는 변수에서 값을 분리한다

 **리스트**
 - 리스트는 크기나 내용을 변경할 수 있다
*리스트는 대괄호나 list함수를 사용해서 생성한다
- +연산자를 이용하면 두 개의 리스트를 합칠 수 있다
- sort 함수를 이용해서 있는 그대로 리스트를 정렬할 수 있다
- start:stop을 지정해서 원하는 크기만큼 잘라낼 수 있다

**딕셔너리**
- 딕셔너리는 키-값쌍을 저장하며 키와 값은 모두 파이썬 객체임
- 각 키가 값과 연관되어, 특정 키가 주어질 때 값을 편리하게 검색, 삽입, 수정 또는 삭제 가능

**집합**
- 집합은 고유한 원소만 담는 정렬되지 않는 자료형
- set함수나 중괄호를 이용해서 생성할 수 있다
